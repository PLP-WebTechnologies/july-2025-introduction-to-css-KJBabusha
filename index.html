About the Box Model

The CSS box model is a fundamental concept that determines how elements are sized and spaced on a web page. It includes content, padding, border, and margin.

Margin

Margin creates space outside the element's border.

Padding

Padding adds space inside the element, between content and border.

Border

Border wraps around the padding and content, defining the edge of the element.
